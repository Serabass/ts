# @декораторы 

**Декораторы** - одно из нововведений ESNext.
По сути своей декоратор - это обычная функция, котоая принимает от одного до трёх аргументов.
В Спецификации TS есть 4 вида декораторов, для которых прописаны тайпинги:

 * ClassDecorator
 * MethodDecorator
 * PropertyDecorator
 * ParameterDecorator
